# Contributing

Thanks for helping keep this list useful. The goal is a *curated* list, not a directory of everything that exists, so the bar is deliberately a little high.

## What gets added

A tool should clear most of these:

- **You'd actually recommend it.** Not "it exists", but "this is one of the good ones for the job".
- **It's reachable for free**, at least for a real task. Freemium is fine — it gets the `$` marker. Tools with no usable free path are usually a no.
- **It's specific.** A focused tool that does one thing well beats another all-in-one suite that overlaps three existing entries.
- **It still works.** Live link, maintained, not abandoned or full of dark patterns.

Bonus points (and a marker) for:

- 🔒 running entirely in the browser with no upload, or
- `OSS` being open source with a public repo.

## What gets rejected

- SEO listicles, link shorteners, and "top 10" content sites.
- Tools that upload your files but advertise themselves as private.
- Near-duplicates that don't add anything over an existing entry.
- Self-promotion with a thin or broken product behind it. A maintainer adding their own genuinely good tool is welcome — just hold it to the same bar as everything else, and don't stack multiple entries for the same product.

## Format

Keep entries in the existing style and **alphabetical within their section**:

```
- [Tool name](https://example.com) 🔒 `OSS` `$` — One sentence, factual, what it does and any real catch (free-tier cap, upload required, platform).
```

- Lead the description with what it *does*, not adjectives.
- Mention the trade-off honestly — a file-size cap, an upload, a platform limit — if there is one.
- One line. If it needs two, it's trying to do too much.
- Drop the markers that don't apply. Most tools have none.

## How

Open a pull request that edits `readme.md`, or an issue if you're not sure it fits. One tool per PR keeps review quick. If you're removing something, say why (dead link, now paywalled, changed behaviour) so the next person doesn't re-add it.
